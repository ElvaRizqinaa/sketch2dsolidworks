# Sketch2dsolidworks
Project ini bertujuan untuk melatih logika menggambar teknik, penggunaan geometric relations, dan dimensi yang presisi. Terdiri dari 5 bagian dengan tingkat kesulitan bertahap.
## Basic SolidWorks CAD Exercises
This repository contains a collection of 3D CAD models created using. The project focuses on mastering 2D sketching, geometric relations, and basic 3D feature creation (Extrude Boss/Base).
## Project Overview
These exercises demonstrate the progression from simple shapes to complex geometries involving tangential curves and angled slots.
* **Software:** SolidWorks [Version, e.g., 2024]
* **Units:** MMGS (millimeter, gram, second)
* **Status:** Completed & Fully Defined 
### Key Features & Learning Outcomes
#### Part 1: The Foundation
* Basic sketching using Lines and Smart Dimensions.
* Understanding the importance of the **Origin point**.
* Creating a fully defined base profile.
#### Part 2: Trimming & Symmetry
* Utilizing **Circle** and **Trim Entities** to create contours.
* Using Centerlines for symmetry and construction.
#### Part 3: Relations & Slots
* Mastering **Sketch Relations** (Tangent, Equal, Vertical).
* Creating **Straight Slots** and handling asymmetrical features.
#### Part 4: Angled Features (The Challenge)
* Defining angled slots using construction lines and degrees.
* Advanced dimensioning techniques to prevent "Over Defined" errors.
* Precise coordinate mapping for angled geometry.
#### Part 5: Complex Curves & Tangency
* "Skeleton" sketching method using coordinates.
* Managing complex **Tangent** relations between arcs and lines.
* Using **Sketch Fillets** for smooth transitions (R3, R5, R10).
